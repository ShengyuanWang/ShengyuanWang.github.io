标签语义

标题标签
<h1> - <h6> 从一级标题到六级标题
<p> 段落标签
<br/> 换行标签

文本格式化标签

加粗标签<strong></strong>
倾斜标签<em></em>
删除线<del></del>
下划线<ins></ins>

div span标签

<div> </div> 独自占一行//大盒子
<span></span> 跨距不换行//小盒子

图像标签

<img src="url" />
alt 替换文本，图像显示不出来的时候用文字代替
title 提示文本，鼠标放到图像上提示的文字
width 设置图像的宽度
height 设置图像高度
border 边框像素

超链接标签

<a href="url" target="method">word</a>
链接的分类
target的默认_self本窗口打开文件 _blank新窗口打开文件
外部链接
内部链接直接里面的名字就可以
空链接里面放一个#就OK
如果这个地址是一个文件或者压缩包，点击之后直接下载文件或者压缩包
中间除了text， 照片视频都可以填进去
矛点链接：href属性值为#名字的形式例如“#two”

表格标签

<table>
        <tr>
            <td>word</td>
            <td>word1</td>
        </tr>
        <tr>
            <td>word2</td>
        </tr>
    </table>

<th></th> 表头单元格 居中加粗
表格属性：
align：left center right 规定表格想对周围元素的对齐方式
border 1 / "" 规定表格单元是否拥有边框
cellpadding 规定单元边沿与其内容之间的空白，默认ipx
cellspacing 规定单元格之间的空白，默认2px
width 规定表格的宽度

合并单元格

rowspan 跨行合并，rowspan=“合并单元格的个数”
colspan 跨列合并， colspan=“合并但与昂哥的个数”

表单标签

目的/收集用户信息
组成/表单域 表单控件 提示信息
<form action="url" method="提交方式" name="表单域名称">
        表单元素控件
</form>
表单元素
<input type="属性"/>
type:
				text 文本框里面可以输入任何文字
				password 密码框 用户看不见输入的密码
				radio 单选按钮 可以实现多选一
name:
				对应的名字，radio必须有同样的名字才能多选一
value：
				默认的值
checked: 
				网页打开的时候默认选上这个地方
maxlength:
				输入的最大的长度

label标签
<label for="text">username</label><input type="text" id="text"/>
下拉表单

  <option>
  </option>
</select>
selected="selected"默认选中
